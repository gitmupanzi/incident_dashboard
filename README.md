# DASHBOARD DE GESTION DES DONNÉES ÉPIDÉMIOLOGIQUES (Streamlit)

Ce projet est un **dashboard Streamlit** pour la gestion, l’analyse et la visualisation de données épidémiologiques (ex. choléra / IDSR), avec :

- nettoyage et standardisation des variables,
- tableaux croisés (province / zone / semaine),
- courbes épidémiologiques et indicateurs (cas, décès, létalité),
- export (CSV/Excel),
- **cartes statiques** (GeoJSON) avec jointure fuzzy (optionnel).
